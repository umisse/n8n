# N8N Application for Appliku

To use this application:
 - go to [Appliku Dashboard](https://app.appliku.com/)
 - Create a server, if you don't have one already
 - Create a custom application:
   - give it a name
   - Repository URL: `https://github.com/appliku/n8n.git`
   - branch: `main`
   - select your server
  
 - After the application is created, wait for it to be initialized and click the "Deploy Now" button.
